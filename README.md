Welcome to the source code of my personal portfolio. This project is a custom-built, high-performance "vCard" style digital resume and project showcase, designed entirely from scratch. 

## 🎯 Project Overview
As a Computer Science and Technology (CST) student, I wanted a portfolio that reflects my technical focus. Instead of relying on heavy frameworks or templates, I engineered this site using core web technologies to ensure maximum speed, total structural control, and a sleek "Industrial Tech" aesthetic.

### Key Features
* **Single Page Application (SPA) Architecture:** Uses Vanilla JavaScript to smoothly switch between sections (About, Resume, Portfolio) without reloading the browser.
* **Modern CSS Grid & Flexbox:** Fully responsive layout utilizing advanced CSS structural properties.
* **Glassmorphism & Custom Theming:** A deep dark mode UI (`#111111`) paired with an Electric Cyan (`#00e5ff`) accent to match the *MI Tech Arsenal* brand.
* **DOM Manipulation:** Custom JavaScript logic handles state changes and CSS animation triggers (`fadeIn`).

## 🛠️ Tech Stack
* **HTML5:** Semantic structure.
* **CSS3:** Custom styling, CSS Variables (`:root`), Grid, Flexbox, and Keyframe Animations.
* **Vanilla JavaScript:** Event listeners and DOM traversal for tab-switching logic.
* **Ionicons:** Professional vector graphics for UI elements.
